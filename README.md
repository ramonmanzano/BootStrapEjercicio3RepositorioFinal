# Ejercicio 3 - Componentes de Bootstrap

## Descripción del Ejercicio
Este ejercicio consiste en la implementación práctica de diversos componentes de Bootstrap 5, demostrando el dominio de elementos interactivos y de navegación esenciales para el desarrollo web moderno.

## Componentes Implementados

### 1. Menús Desplegables (Dropdowns)
**Captura de pantalla del componente:**
![Menús Desplegables](https://github.com/ramonmanzano/BootStrapEjercicio3RepositorioFinal/blob/0061f81ef7d84f8524ecb166f99e1faf7fc461ad/img/ej1.png)

**Características implementadas:**
- Varios botones dropdown con diferentes colores (primary, success, dark)
- Diferentes direcciones de despliegue: dropdown (abajo), dropup (arriba), dropstart (izquierda), dropend (derecha)
- Menús desplegables con listas de elementos interactivos
- Ejemplo de dropdown con ancho personalizado (280px)
- Uso de clases `btn-group` para agrupación de botones

**Funcionalidad:**
Cada botón dropdown muestra una lista de opciones al hacer clic, implementando la interactividad completa que Bootstrap ofrece para estos componentes.

### 2. Grupos de Listas (List Groups)
**Captura de pantalla del componente:**
![Grupos de Listas](https://github.com/ramonmanzano/BootStrapEjercicio3RepositorioFinal/blob/0061f81ef7d84f8524ecb166f99e1faf7fc461ad/img/ej2.png)

**Características implementadas:**
- Listas básicas con elementos activos e inactivos
- Lista numerada con `list-group-numbered`
- Listas con elementos de diferentes colores (info, secondary, success, danger, warning, light)
- Elementos con badges y contadores
- Elementos con checkboxes interactivos
- Cards integradas mostrando artículos con títulos, contenido y fecha

**Estructura visual:**
Las listas están organizadas en un sistema de grid responsivo, mostrando múltiples variantes de este componente Bootstrap en una sola página.

### 3. Ventanas Modal
**Captura de pantalla del componente:**
![Ventanas Modal](https://github.com/ramonmanzano/BootStrapEjercicio3RepositorioFinal/blob/0061f81ef7d84f8524ecb166f99e1faf7fc461ad/img/ej3.png)

**Características implementadas:**
- Modal básico con cabecera, cuerpo y pie
- Modal con contenido desplazable (`modal-dialog-scrollable`)
- Texto de ejemplo con múltiples párrafos para demostrar el scroll
- Botones de acción en el pie del modal (Cancelar y Aceptar)
- Animación de fade al abrir y cerrar el modal

**Funcionalidad:**
La ventana modal se activa mediante un botón y proporciona una capa de interacción sobre el contenido principal, ideal para formularios, confirmaciones o información adicional.

### 4. Menús de Navegación y Pestañas
**Captura de pantalla del componente:**
![Navegación y Pestañas](https://github.com/ramonmanzano/BootStrapEjercicio3RepositorioFinal/blob/0061f81ef7d84f8524ecb166f99e1faf7fc461ad/img/ej4.png)

**Características implementadas:**

**Navegación horizontal:**
- Alineación izquierda (default)
- Alineación centrada (`justify-content-center`)
- Alineación derecha (`justify-content-end`)

**Navegación vertical:**
- Menú vertical con `flex-column`

**Sistema de pestañas (Tabs):**
- Pestañas con contenido intercambiable
- Transiciones suaves entre pestañas
- Integración de píldoras de navegación (`nav-pills`) dentro de las pestañas

**Funcionalidad:**
Los sistemas de navegación permiten organizar el contenido en secciones accesibles, mientras que las pestañas proporcionan una forma eficiente de mostrar contenido relacionado en un espacio limitado.

## Tecnologías Utilizadas

- **Bootstrap 5.3.2** (via CDN)
- **HTML5** semántico
- **CSS3** (incluido en Bootstrap)
- **JavaScript** (incluido en Bootstrap Bundle)

## Cómo Visualizar los Ejercicios

1. **Método directo:** Abrir cada archivo HTML individualmente en un navegador web moderno
2. **Interactividad:** Todos los componentes son completamente interactivos
   - Los dropdowns responden a clics
   - Las pestañas cambian de contenido
   - El modal se abre/cierra con animaciones
   - Los checkboxes en list groups son funcionales

## Criterios de Implementación Cumplidos

✓ **Dropdowns:** Múltiples variantes y direcciones implementadas  
✓ **List Groups:** Diferentes tipos y estilos incluidos  
✓ **Modal:** Funcionalidad completa con scroll y acciones  
✓ **Navegación:** Sistemas horizontal, vertical y pestañas  
✓ **Responsividad:** Todos los componentes son responsive  
✓ **Accesibilidad:** Atributos ARIA y semántica adecuada
